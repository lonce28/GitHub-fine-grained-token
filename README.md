# 中国工程行业技术博客

这是一个最小可用的静态博客网站，适合部署到 GitHub Pages 或 Cloudflare Pages。

## 推荐免费域名与托管

- GitHub Pages：默认域名形如 `https://用户名.github.io/仓库名/`
- Cloudflare Pages：默认域名形如 `https://项目名.pages.dev/`

正式运营建议购买独立域名，例如 `工程技术网.com`、`pipeline-tech.cn`、`hse-tech.cn` 一类，并根据中国大陆访问和百度收录需求评估备案、服务器位置和 CDN。

## 页面

- `index.html`：首页
- `articles/index.html`：文章列表页
- `articles/hse-risk-control.html`：文章详情页示例
- `categories/index.html`：分类页
- `about.html`：关于我们页
- `sitemap.xml`：搜索引擎站点地图
- `robots.txt`：爬虫访问规则
